Projects

Automated secure vault room

Level 1 | Hardware project | 2020 | Project Leader

The system was ideally developed to be used in banks and it comprises of Dual fingerprint match, bidirectional visitor counter, time constraint using RTC module and security alert during any kind of unauthorized access. Through our system, all regulations of bank can be monitored automatically. For e.g.- Allowing only two authorized people at a time (One person also not allowed), access is allowed only once per hour.

System Connecting Service Provider And Service Consumer In Construction And Event Management

Level 2 | Software Project | 2020 | Leader | OnGoing

Service providers can register to our system, while registering they have to upload the documents and NIC confirming their qualifications. These documents will be verified by third party interface. Service consumers can able to search for a service provider, view their profile, chat with them and able to book them. Search results will be displayed according to filtration based on nearest location. Service provider able to accept or reject the request and able to send the quotation. Both parties can able to make complaint to the admin. We are currently developing the system using MERN stack in web portal and react native in mobile app.
